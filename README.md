# 🔐 File Hashing and Integrity Verification Tool

A Python-based desktop application for verifying file integrity using cryptographic hashing algorithms. The application provides multiple modules including file hashing, hash comparison, checksum verification, directory monitoring, and VirusTotal malware scanning.

---

## 📌 Features

- Generate cryptographic hashes for files
- Supports multiple hashing algorithms
  - MD5
  - SHA-1
  - SHA-256
  - SHA-384
  - SHA-512
  - SHA3 Family
- Compare file hash with a known hash value
- Compare two files using hash values
- Generate and verify checksum files
- Monitor directories for added, modified, and deleted files
- VirusTotal hash-based malware scanning
- Simple Tkinter-based graphical interface

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/gaurav720p/file-hashing-and-integrity-verification-tool.git
```

Go into the project folder

```bash
cd file-hashing-and-integrity-verification-tool
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
